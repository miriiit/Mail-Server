# Express.js and EJS

Example of using Node.js, Express.js and EJS in Mailing Server.

### Endpoints

+ `/`
+ `/mail`
+ `/mail/send`

## Install

`npm install`

## Run

1. `npm run dev`
2. Open `http://localhost:8080` in a web browser.

### Structure
+ `Routes`
+ `Controller`
+ `middleware`
+ `config`
